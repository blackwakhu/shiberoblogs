How to add Tailwind CSS to a vite project

Derrick Shibero Wakhu May 21, 2025

Tailwind CSS is a CSS framework that utilizes classes with CSS code that only represent a singular CSS style rule. 

The main advantage of it is that it allows a higher degree of freedom to the programmer. This is as compared to some frameworks like bootstrap that offer generic code for each element. The main drawback of Tailwind is the use of many classes to achieve a single style that could be achieved by a dedicated CSS rule. This results in large code that feel too big to debug. This results to HTML code that looks like the following

<!doctype html>
<html>
    <head>
        <meta charset="UTF-8">  
        <meta name="viewport" content="width=device-width, initial-scale=1.0">  
        <link href="/src/styles.css" rel="stylesheet">
    </head>
    <body>  
        <h1 class="text-3xl font-bold underline">    
            Hello world!  
        </h1>
    </body>
</html>

How to start

The following installation will be applicable with any framework installed using vite js. This include frameworks like Laravel, SvelteKit, React, Router, Nuxt and SolidJS. 

1. Install tailwind css in the folder containing the project

npm install tailwindcss @tailwindcss/vite

2. Configure the vite plugin by altering the vite.config.js by adding the following

import tailwindcss from '@tailwindcss/vite'
export default defineConfig({
    plugins: [tailwindcss()],
})

The following is how it will look like in react project

import { defineConfig } from 'vite'
import tailwindcss from '@tailwindcss/vite'
import react from '@vitejs/plugin-react'


export default defineConfig({
  plugins: [
        react(), 
        tailwindcss(),
      ],
})

3. Add the following on top of the css file

@import "tailwindcss";

4. Start using in your project. An example in a react project

<div>
    <h2 className="text-xl font-semibold text-gray-800 mb-4">
        Dashboard
    </h2>
</div>
