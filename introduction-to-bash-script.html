Introduction to Bash Script

Derrick Shibero Wakhu May 28, 2025

One of the first thing one learns when they start using a Linux OS distribution, they learn how to use the terminal. The terminal is available in different  operating systems. In Linux, the default is bash(Bourne Again Shell), in Mac OS it is zsh and in Windows OS it is the Powershell. In Linux, the terminal is one of the most important things to learn. Some of the commands in the shell include:

  • cat - for displaying contents in a file
  • ls - for listing contents in directory
  • sudo - allows the user access root privileges
  • touch - creates a new file
  • cd - changes the directory
  • find - search for an item in a directory

In order to automate some commands bash allows users to create bash files. In this blog we are going to focus in bash files. 

In order to start we create a bash file with the extension .sh. Using sudo command we are going to make the file executable by running the following command on the terminal. Replace the script-file.sh with your own file name.

sudo chmod +x script-file.sh

When running the above file you do the following

./script-file.sh

In the following we are going to focus on displaying the results, variables and functions. 

Displaying results

The command used here is echo. Examples include:

echo "Hello world";

Variables

The following way is how you initialise variables and do some simple arithmetic with them:

# declaring and assigning a variable
phrase="I love bash scripting"
echo $phrase

Bash is not designed to do arithmetic. In order to perform such you add the expr function before the equation.

# declaring, assigning integer variables and doing some math
x=1
y=2
z=`expr $x + $y`
echo "$x + $y = $z"

Comparisons

We will begin by string comparisons

x="hello"
y="hello"

if [ $x=$y ]
then
    echo "x and y match"
elif [ $x!=$y ]
then 
    echo "x and y dont match"
fi

Now integer comparisons

x=1
y=2

if [ $x -eq $y ] #equality
then 
    echo "they are both equal"
elif [ $x -gt $y ] # for greater than; use -lt for less than
then
    echo "$x is greater than $y"    
else
    echo "$x is less than $y"
fi

Functions

They can be made the following way

# creating the function
hello(){
    echo "hello World"
}

# calling the function
hello

Function with parameters

# creating the function
helloName(){
    echo "Hello $1"
}

name="John Doe"

helloName $name
